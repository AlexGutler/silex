APIS E SILEX - Projeto Fase 1

Preparando o ambiente

O objetivo geral desse projeto será desenvolver uma API REST, utilizando inclusive banco de dados. Para isso será necessário termos uma estrutura básica necessária configurada para iniciarmos a criação das APIs.

Nessa fase do projeto, você instalará o Silex e criará 1 rotas principal, apenas para garantir que tudo está configurado e funcionando.

1) Rota: /clientes

Com a rota /clientes, faça a simulação da listagem de clientes com Nome, Email e CPF/CNPJ vindo de um array. O formato de exibição deve ser json.
